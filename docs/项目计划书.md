# MostBox 项目计划书

> 纯 P2P、零知识、无中心管控的分布式付费存储网络

---

## 项目摘要

| 项目         | 内容                                                          |
| ------------ | ------------------------------------------------------------- |
| **名称**     | MostBox                                                       |
| **代币**     | USDC/USDT（支付）+ MOST（治理+价值捕获）                      |
| **核心定位** | 用户拥有数据、节点贡献资源、代码执行规则的下一代存储基础设施  |
| **技术底座** | Hyperswarm + Hyperdrive + Base 智能合约 + ERC-4337            |
| **当前状态** | v0.1.0 已实现 P2P 文件分享、聊天频道、Web3 工具箱、桌面客户端 |

---

## 一、市场痛点

### 1.1 传统网盘

| 问题       | 表现                                   |
| ---------- | -------------------------------------- |
| 数据控制权 | 平台可随时删除、审查、限制访问         |
| 隐私风险   | 服务商可扫描文件内容，存在数据泄露隐患 |
| 体验限制   | 限速、限流、空间付费昂贵               |

### 1.2 现有去中心化存储

| 项目     | 核心缺陷                                     |
| -------- | -------------------------------------------- |
| Filecoin | 半中心化，节点需专业矿机，存储方可见明文哈希 |
| Storj    | 依赖以太坊半中心化架构，节点门槛中等         |
| Sia      | 自研链生态封闭，分片信息对存储方可见         |

### 1.3 市场空白

**缺乏纯 P2P 原生、存储方零知情权、人人可参与的通用型付费存储网络。**

---

## 二、核心价值主张

| 价值             | 实现方式                                                  |
| ---------------- | --------------------------------------------------------- |
| **绝对隐私**     | 客户端 AES-256-GCM 加密后上传，节点仅见密文，元数据全加密 |
| **完全去中心化** | 无中心服务器，规则由智能合约执行，无法篡改或关停          |
| **零门槛参与**   | 浏览器即可存储文件，普通电脑可部署节点赚取收益            |
| **高效低成本**   | P2P 直连无中转，资源按需定价，比传统网盘低 30%-50%        |
| **无感体验**     | Base + ERC-4337 Paymaster 代付 Gas，用户无需持有 ETH      |
| **稳定支付**     | 稳定币（USDC/USDT）支付，无价格波动风险                   |

---

## 三、网络角色

### 3.1 存储用户

- 使用稳定币（USDC/USDT）支付存储费用
- 持有文件私钥，随时取回、分享
- 可设置分享权限，支持付费下载

### 3.2 存储节点（矿工）

- 贡献硬盘空间和带宽
- 自动接收订单，托管加密分片
- 赚取稳定币收益（无价格波动风险）
- 需质押 MOST 作为保证金

### 3.3 社区治理

- MOST 持有者投票决定网络参数（手续费、质押要求）
- 审核技术升级提案
- 无特殊权限，无法干预数据或运营

## 四、经济模型

### 4.1 双轨经济

| 代币       | 角色          | 用途                           |
| ---------- | ------------- | ------------------------------ |
| **稳定币** | 支付媒介      | 用户支付存储费用，节点获得收益 |
| **MOST**   | 治理+价值捕获 | 节点质押、社区投票、价值增值   |

### 4.2 MOST 代币属性

| 属性     | 值                        |
| -------- | ------------------------- |
| 名称     | MostBox Token             |
| 符号     | MOST                      |
| 总供应量 | 21,000,000 枚（固定上限） |
| 链       | Base（ERC-20）            |

### 4.3 MOST 分配方案

| 用途           | 比例 | 释放规则                 |
| -------------- | ---- | ------------------------ |
| 节点质押激励池 | 40%  | 早期节点招募激励         |
| 团队           | 10%  | 4 年线性解锁             |
| 社区生态       | 30%  | 空投/开发者基金/节点招募 |
| 储备           | 20%  | 流动性、做市             |

### 4.4 定价与收益

| 机制            | 规则                                                             |
| --------------- | ---------------------------------------------------------------- |
| **稳定币定价**  | $0.1/GB/月（比传统网盘低 30%-50%）                               |
| **预付费钱包**  | 用户充值稳定币到 MostBox 钱包，余额随时可转出到其他地址          |
| **按 CID 计费** | 上传时一次性扣除存储费用（容量×时长×$0.1/GB/月），已下单不可退款 |
| **开放续费**    | 任何人可用稳定币为任何 CID 续费，延长存储时间                    |
| **收益分配**    | 节点 90% / 网络手续费 10%                                        |
| **节点结算**    | 每月结算一次，按用户实际存储量分配                               |
| **节点等级**    | 按在线时长、存储量、信誉分级，高等级优先匹配高价值订单           |
| **惩罚机制**    | 离线 >24h 扣当日收益；丢失数据扣质押 MOST；多次违规永久封禁      |

### 4.5 存储订单模型

```
用户充值稳定币 → MostBox 钱包余额增加
      ↓
上传文件 → 选择存储时长 → 一次性扣除费用（容量×时长×$0.1/GB/月）
      ↓
节点接单存储 → 按月从激励池领取稳定币奖励
      ↓
任何人可为任何 CID 续费 → 激励池增加 → 存储时间延长
```

| 特性           | 说明                                                     |
| -------------- | -------------------------------------------------------- |
| **预付费钱包** | 用户余额随时可转出到其他钱包地址，已下单费用不可退款     |
| **CID 即权限** | 知道 CID 即可访问和续费文件，符合 P2P 信息自由流动原则   |
| **共享激励池** | 同一 CID 的所有续费资金合并，节点按存储时间比例分配      |
| **版本独立**   | 文件修改生成新 CID，需创建新存储订单，旧版本订单继续有效 |
| **到期处理**   | 单个 CID 资金耗尽 → 该文件从节点移除，不影响其他 CID     |

### 4.6 MOST 价值捕获

| 机制           | 规则                                                          |
| -------------- | ------------------------------------------------------------- |
| **手续费销毁** | 存储订单手续费 50% 从 DEX 购买 MOST 并销毁                    |
| **罚没销毁**   | 节点质押罚没 100% 销毁                                        |
| **长期效应**   | 网络使用越多 → 销毁越多 MOST → MOST 越稀缺 → 节点质押价值越高 |

### 4.7 正向循环

```
用户支付 USDC → 节点获得 90% 稳定币收入
                     ↓
               10% 手续费 → 50% 购买并销毁 MOST
                     ↓
               MOST 总量减少 → MOST 增值
                     ↓
               节点质押 MOST 更有价值 → 更多节点加入
                     ↓
               网络容量增加 → 更多用户 → 更多销毁
```

## 五、竞品对比

| 项目        | 去中心化             | 存储方知情权     | 节点门槛             | 隐私强度 | 支付方式      |
| ----------- | -------------------- | ---------------- | -------------------- | -------- | ------------- |
| Filecoin    | 半去中心化           | 可见明文哈希     | 高（专业矿机）       | 中       | FIL           |
| Storj       | 半去中心化           | 可见文件哈希     | 中                   | 中       | USDC/信用卡   |
| Sia         | 半去中心化           | 可见分片信息     | 中                   | 中       | Siacoin       |
| MaidSafe    | 高度去中心化         | 零知情权         | 低                   | 高       | 网络积分      |
| **MostBox** | **近乎完全去中心化** | **绝对零知情权** | **极低（普通电脑）** | **顶级** | **USDC+MOST** |

### 差异化优势

1. **纯 P2P 原生** — 无中心化组件，抗审查、抗宕机
2. **绝对隐私** — 元数据全加密，存储方零知情
3. **极低门槛** — 浏览器即用，普通电脑可当节点
4. **稳定支付** — 稳定币支付，用户无价格波动风险
5. **开放续费** — 任何人可为任何文件续费，公共价值由社区共同维护

## 八、总结

MostBox 基于 Holepunch 生态的纯 P2P 技术，打造真正去中心化、隐私优先的付费存储网络。用户真正拥有数据，任何人可通过贡献资源获得收益。

通过稳定币+MOST 双轨经济模型，MostBox 既提供用户友好的稳定支付体验，又通过 MOST 代币实现去中心化治理和价值捕获。用户按 CID 购买存储空间，任何人可为任何文件续费，网络使用推动 MOST 销毁与增值，形成健康的正向循环。
