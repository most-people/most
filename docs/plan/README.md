## 当前 MVP 口径

首版只验证一个闭环：

```
CID
  → most:// 分享链接
  → 发布者按 CID topic 做种
  → 下载者凭链接下载并重算 CID 校验
  → 下载者默认持续做种
  → 发布者退出后，只要还有下载者在线即可继续传播
```

以下能力明确不进入 MostBox 文件分享主线：上链存储协议、智能合约、USDT 支付、质押、订单、赏金猎人、fraud proof、Treasury、云端下单/上传。

Web3 产品入口和以太坊钱包工具保留为独立工具箱，不作为 P2P 发布、下载或做种的前置条件。
