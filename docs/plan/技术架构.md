# MostBox 技术架构

> MostBox v1 是 P2P 分享工具：`most://` 链接定位文件，Hyperswarm 发现 peer，Hyperdrive 保存完整副本，下载后重算 CID 校验完整性。

---

## 一、分层架构

```
┌─────────────────────────────────────────────────────────────┐
│                      用户交互层                              │
│  Web demo  │  Electron 桌面端  │  做种管理台  │  Web3 工具箱 │
├─────────────────────────────────────────────────────────────┤
│                      接口自动化层                            │
│  HTTP API  │  WebSocket 事件  │  OpenAPI 文档  │  错误码       │
├─────────────────────────────────────────────────────────────┤
│                      应用服务层                              │
│  文件管理  │  分享链接  │  下载任务  │  节点设置  │  聊天频道  │
├─────────────────────────────────────────────────────────────┤
│                      内容完整性层                            │
│  UnixFS CID v1  │  内容寻址  │  本地重算 CID 校验  │
├─────────────────────────────────────────────────────────────┤
│                      P2P 网络层                              │
│  Hyperswarm 4.x  │  Hyperdrive 13.x  │  Corestore 7.x │
└─────────────────────────────────────────────────────────────┘
```

无区块链层。Web3 工具箱保留为独立页面，不参与文件分享协议。

## 二、技术栈

| 层级 | 技术 | 作用 |
| ---- | ---- | ---- |
| 网络层 | Hyperswarm 4.x | 节点发现、NAT 打洞、P2P 组网 |
| 存储层 | Hyperdrive 13.x + Corestore 7.x | 完整文件副本存储、复制 |
| 校验层 | UnixFS CID v1 | 文件身份、完整性校验 |
| 接口层 | Hono API + WebSocket + OpenAPI | 桌面端、管理台、脚本共用 |
| 交互层 | Next.js 16 + React 19 | Web demo、桌面端 UI、管理台 |
| 桌面层 | Electron 41 | 正式发布、下载、做种入口 |
| Web3 工具 | ethers.js | 独立钱包/密钥派生工具箱 |

### CID 生成参数

MostBox 复用当前 `server/src/core/cid.js` 实现：`ipfs-unixfs-importer@16.1.5`、`cidVersion: 1`、`rawLeaves: true`、`wrapWithDirectory: false`。升级依赖前必须通过 CID 黄金样本测试。

### 分享链接

`most://<cid>?filename=...` 是唯一 MVP 分享格式。

| 字段 | 说明 |
| ---- | ---- |
| `cid` | UnixFS CID v1，文件身份和 topic 来源 |
| `filename` | 用户可见文件名，只用于保存和展示 |

### 接口优先原则

做种能力先沉到 HTTP API 与 WebSocket 事件层，再由不同入口复用：

| 入口 | 定位 | 优先级 |
| ---- | ---- | ------ |
| 桌面端 | 普通用户发布、下载、做种 | P0 |
| Web 管理台 | 人类查看做种 daemon 状态 | P0 |
| OpenAPI/API 文档 | 脚本和第三方集成 | P0 |
| 薄 CLI | 安装、服务生命周期、诊断 | P1 |


## 五、非 MVP 能力

| 能力 | 当前决策 |
| ---- | -------- |
| 上链存储协议 | 不实现 |
| 智能合约 / USDT / 质押 / 结算 | 不实现 |
| 订单、报价、手动选节点 | 不实现 |
| 赏金猎人 / fraud proof / Treasury | 不实现 |
| 云端上传或托管 | 不实现 |
| 默认应用层加密 | 暂不内置；用户可自行本地加密后分享 |
| 浏览器长期做种 | 暂不承诺；正式能力使用桌面端/daemon |
