# MostBox 技术架构

> MostBox 是 P2P 文件分享工具：`most://` 链接定位文件，Hyperswarm 发现 peer，Hyperdrive 保存完整副本，下载后重算 UnixFS CID v1 校验完整性。

## 一、分层架构

```
┌─────────────────────────────────────────────────────────────┐
│                      用户交互层                              │
│  Web demo  │  Electron 桌面端  │  做种管理台  │  Web3 工具箱 │
├─────────────────────────────────────────────────────────────┤
│                      接口自动化层                            │
│  HTTP API  │  WebSocket 事件  │  OpenAPI 文档  │  错误码       │
├─────────────────────────────────────────────────────────────┤
│                      应用服务层                              │
│  文件管理  │  分享链接  │  下载任务  │  节点设置  │  做种持有  │
├─────────────────────────────────────────────────────────────┤
│                      内容完整性层                            │
│  UnixFS CID v1  │  内容寻址  │  本地重算 CID 校验  │
├─────────────────────────────────────────────────────────────┤
│                      P2P 网络层                              │
│  Hyperswarm 4.x  │  Hyperdrive 13.x  │  Corestore 7.x │
└─────────────────────────────────────────────────────────────┘
```

没有区块链协议层。Web3 工具箱保留为独立页面，不参与文件分享、下载或做种协议。

## 二、核心技术决策

| 主题 | 决策 |
| ---- | ---- |
| 文件身份 | UnixFS CID v1 |
| CID 参数 | `cidVersion: 1`、`rawLeaves: true`、`wrapWithDirectory: false` |
| 分享格式 | `most://<cid>?filename=...` |
| topic 规则 | 使用 `cid.multihash.digest`，不额外 hash、不截断 |
| Hyperdrive key | 固定为 `/<cid>` |
| 文件模型 | 完整副本，不做分片、纠删码或云端托管 |
| 权限模型 | CID 即权限，知道链接即可尝试下载 |
| 默认加密 | 不做，私密文件由用户先本地加密 |
| 做种默认值 | 发布和下载成功后固定自动做种 |

升级 `ipfs-unixfs-importer` 前必须跑 CID 黄金样本测试。

## 三、接口优先原则

做种能力先沉到 HTTP API 与 WebSocket 事件层，再由桌面端、管理台和脚本复用。

| 入口 | 定位 | 当前状态 |
| ---- | ---- | -------- |
| 桌面端 | 普通用户发布、下载、做种 | 主入口 |
| Web 管理台 | 查看 daemon 状态、holding、日志和设置 | 基础完成 |
| OpenAPI | 脚本和第三方集成 | 基础完成 |
| 薄 CLI | 安装、服务生命周期、诊断 | 后续 |
| Web demo | 展示和轻量体验 | 不承诺长期做种 |

```

```

## 五、当前待打磨模块

| 模块 | 优先级 | 目标 |
| ---- | ------ | ---- |
| Quick Start | P0 | 发布者、下载者、daemon 三条路径可照做 |
| 管理台安全门槛 | P0 | 默认本机访问，或公网暴露时有口令/警告策略 |
| 主应用失败提示 | P0 | 无 peer、校验失败、超时、取消都可读 |
| 真实网络长测 | P0 | 100MB / 1GB 文件传播和重启恢复 |
| 种子健康度 | P1 | 展示 peer 数、最近连接、最后 serve 时间 |
| 同名文件冲突 | P1 | 下载保存前给出明确选择 |
| 薄 CLI | P1 | 初始化、启动、停止、诊断 |
| 单文件暂停/恢复 | P2 | 不改变默认自动做种，只提供单文件级管理 |
| `magnet:` 兼容 | P3 | 后续独立评估 |

## 六、非 MVP 能力

| 能力 | 当前决策 |
| ---- | -------- |
| 上链存储协议 | 不实现 |
| 智能合约 / USDT / 质押 / 结算 | 不实现 |
| 订单、报价、手动选节点 | 不实现 |
| 赏金猎人 / fraud proof / Treasury | 不实现 |
| 云端上传或托管 | 不实现 |
| 默认应用层加密 | 暂不内置；用户可自行本地加密后分享 |
| 浏览器长期做种 | 暂不承诺；正式能力使用桌面端或 daemon |
